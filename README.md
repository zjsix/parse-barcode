# parse-barcode
识别图片中的条形码并返回
由于微信小程序识别条形码会不准确，<a href="https://developers.weixin.qq.com/community/develop/doc/00048e1ee981b0a500de1ed0f59414"> 官方不作为</a>
只能想了这么个办法，小程序拍摄图片转base64发给服务器解析条形码并返回barCode
